# 光博光电企业官网 - 规格说明书

## 1. 项目概述

- **项目名称**: 光博光电企业官网
- **项目类型**: 响应式企业展示网站
- **核心功能**: 展示光博光电公司形象、产品服务，提供联系方式
- **目标用户**: 潜在客户、合作伙伴、求职者

## 2. UI/UX 规格

### 2.1 布局结构

**页面组成**
- 固定顶部导航栏
- 首页: 英雄区 + 核心优势 + 产品预览 + 关于我们预览 + 底部CTA
- 关于我们: 公司简介 + 发展历程 + 企业愿景
- 产品服务: 产品分类 + 产品展示 + 定制服务
- 联系我们: 联系信息 + 在线表单 + 地图

**响应式断点**
- 移动端: < 768px
- 平板: 768px - 1024px
- 桌面: > 1024px

### 2.2 视觉设计

**配色方案**
- 主色: #0A1628 (深邃藏青)
- 辅助色: #1E3A5F (科技蓝)
- 强调色: #00D4AA (活力青绿)
- 背景色: #F8FAFC (纯净白)
- 文字主色: #1A1A2E (墨黑)
- 文字次色: #64748B (岩石灰)

**字体**
- 标题: "Noto Serif SC", serif (优雅中文衬线)
- 正文: "Noto Sans SC", sans-serif (清晰中文无衬线)
- 英文点缀: "Outfit", sans-serif

**间距系统**
- 基础单位: 8px
- 区块间距: 80px (桌面) / 48px (移动)
- 内边距: 24px / 32px / 48px

**视觉效果**
- 卡片: 微妙阴影 `0 4px 24px rgba(10, 22, 40, 0.08)`
- 按钮悬停: 上移 + 阴影增强
- 滚动淡入动画: 透明度 0→1 + Y轴位移 30px→0

### 2.3 组件设计

**导航栏**
- 固定顶部，背景模糊效果
- Logo 左侧，公司名称
- 导航链接右侧，带下划线hover效果
- 移动端: 汉堡菜单

**按钮**
- 主按钮: 强调色背景，白色文字，圆角8px
- 次按钮: 透明背景，边框，悬停填充
- 尺寸: padding 16px 32px

**卡片**
- 白色背景，圆角16px
- 悬停: 轻微上浮 + 阴影加深
- 过渡: 0.3s ease

**表单**
- 输入框: 简洁边框，聚焦时强调色边框
- 标签: 字段上方显示

## 3. 功能规格

### 3.1 首页
- 英雄区: 全屏背景 + 大标题 + 副标题 + CTA按钮
- 核心优势: 4个优势卡片，图标+标题+描述
- 产品预览: 3个产品卡片，点击跳转产品页
- 关于我们预览: 大图+文字介绍
- 底部CTA: 醒目的行动号召

### 3.2 关于我们
- 公司简介: 文字+图片并排
- 发展历程: 时间线形式展示
- 企业愿景: 愿景声明

### 3.3 产品服务
- 产品分类标签
- 产品网格展示
- 定制服务说明

### 3.4 联系我们
- 联系方式卡片: 电话/邮箱/地址
- 在线表单: 姓名/电话/邮箱/留言
- 提交按钮

### 3.5 动画效果
- 页面滚动: 元素渐入动画
- 按钮: hover微动效
- 导航: 平滑滚动
